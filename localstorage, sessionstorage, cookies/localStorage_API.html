Best Practices & Notes:

  • Storage Limit: সাধারণত ৫-১০ এমবি পর্যন্ত ডাটা রাখা যায়।
  • Data Type: শুধুমাত্র String সেভ করা যায়। Object সেভ করতে JSON.stringify() ব্যবহার করুন।
  • Security: সেনসিটিভ ডাটা (Password/Token) এখানে রাখা উচিত নয় কারণ এটি XSS অ্যাটাক এর মাধ্যমে চুরি হতে পারে।
  • Persistence: ব্রাউজার বন্ধ করলেও ডাটা ডিলিট হয় না।
